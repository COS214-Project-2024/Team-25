\doxysubsection{Detailed Description}
Abstract class representing an industrial building. 

The \doxylink{class_industrial}{Industrial} class is derived from the \doxylink{class_building}{Building} class and serves as a base class for various types of industrial buildings. It includes attributes specific to industrial buildings, such as carbon footprint and capacity.
