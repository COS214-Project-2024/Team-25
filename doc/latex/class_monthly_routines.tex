\doxysubsection{Detailed Description}
\doxylink{class_concrete}{Concrete} invoker class that manages monthly government routines. 

This class manages the execution of various transport and building routines for the city. It holds a reference to the \doxylink{class_city_growth}{City\+Growth} object and maintains lists of commands to be executed.
