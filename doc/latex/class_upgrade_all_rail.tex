\doxysubsection{Detailed Description}
\doxylink{class_concrete}{Concrete} command class that upgrades all rail systems. 

This command class encapsulates the action of upgrading all rail systems in the city. It holds a vector of \doxylink{class_rail_system_adapter}{Rail\+System\+Adapter} pointers representing the rail systems to be upgraded.
