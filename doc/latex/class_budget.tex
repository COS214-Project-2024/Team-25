\doxysubsection{Detailed Description}
Singleton class representing the financial budget of the city. 

The \doxylink{class_budget}{Budget} class manages the city\textquotesingle{}s financial resources, including cash and debt. It follows the Singleton design pattern to ensure that only one instance of this class exists throughout the application.
