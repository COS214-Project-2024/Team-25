\doxysubsection{Detailed Description}
Represents a sector within the city containing buildings. 

The \doxylink{class_city_sector}{City\+Sector} class manages a collection of buildings in a specific area of the city. It provides methods to add buildings, retrieve the number of buildings, and access the list of buildings in the sector.
