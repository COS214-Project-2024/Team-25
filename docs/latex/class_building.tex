\doxysubsection{Detailed Description}
Abstract base class representing a generic building. 

The \doxylink{class_building}{Building} class serves as a base class for various types of buildings (e.\+g., \doxylink{class_residential}{Residential}, \doxylink{class_commercial}{Commercial}). It includes essential attributes such as name, type, number of rooms, area, value, and lists of citizens. Derived classes must implement the pure virtual functions to define specific behaviors.
