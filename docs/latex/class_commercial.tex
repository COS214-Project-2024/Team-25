\doxysubsection{Detailed Description}
Abstract class representing a commercial building. 

The \doxylink{class_commercial}{Commercial} class is derived from the \doxylink{class_building}{Building} class and serves as a base class for various types of commercial buildings. It includes attributes specific to commercial buildings, such as capacity and the number of floors.
