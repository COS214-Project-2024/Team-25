\doxysubsection{Detailed Description}
Singleton class representing energy resources in the city. 

The \doxylink{class_energy}{Energy} class manages different types of energy sources such as coal, hydro, wind, and solar. It provides access to their wattage values and follows the Singleton design pattern to ensure that only one instance of this class exists throughout the application.
