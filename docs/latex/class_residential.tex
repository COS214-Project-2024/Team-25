\doxysubsection{Detailed Description}
Abstract class representing a residential building. 

The \doxylink{class_residential}{Residential} class is derived from the \doxylink{class_building}{Building} class and serves as a base class for different types of residential buildings. It includes attributes specific to residential buildings, such as bedrooms, bathrooms, and capacity.
