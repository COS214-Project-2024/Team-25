\doxysubsection{Detailed Description}
\doxylink{class_concrete}{Concrete} command class that renovates all buildings. 

This command class encapsulates the action of renovating all buildings in the city. It holds a vector of \doxylink{class_building}{Building} pointers to operate on during execution.
