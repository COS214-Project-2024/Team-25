\doxysubsection{Detailed Description}
Abstract class representing an institutional building. 

The \doxylink{class_instatutional}{Instatutional} class is derived from the \doxylink{class_building}{Building} class and serves as a base class for various types of institutional buildings. It includes attributes specific to institutions, such as capacity.
