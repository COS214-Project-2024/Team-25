\doxysubsection{Detailed Description}
Abstract base class representing a citizen. 

The \doxylink{class_citizen}{Citizen} class serves as a base class for various types of citizens, encapsulating common properties such as residence, workplace, and satisfaction levels. This class defines the interface for all citizen types.
