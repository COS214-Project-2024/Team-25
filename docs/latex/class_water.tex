\doxysubsection{Detailed Description}
Singleton class representing water resources in the city. 

The \doxylink{class_water}{Water} class manages the water supply in the city, tracking the total amount of water available in liters. It follows the Singleton design pattern to ensure that only one instance of this class exists throughout the application.
